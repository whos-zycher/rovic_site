Rovic

Automatic real-time scam image detection for Discord. Watches every image, checks signatures, bans scammers. No false positives. No stored files.

Pipeline

  1. Ingest: Downloads raw bytes from Discord CDN to memory.
  2. Hash: DCT perceptual hash. Bytes freed instantly.
  3. Lookup: BK-Tree signature check.
  4. Execute: Instant ban or mod review queue.
Distance Status Action
<= 5 Match Deleted, user banned, 7d wipe.
6 - 15 Warn Deleted, flagged for review.
> 15 Safe Ignored. Zero false positives.

Specs

  • Dedupe: LRU MD5 skip. Nullifies spam wave costs.
  • Resilience: Survives JPEG compress, rescale, screenshots.
  • Sync: Redis Pub/Sub global updates.

Privacy

  • [+] 64-bit hashes of confirmed scams in DB.
  • [-] NO images saved to disk.
  • [-] NO message content logged.
  • [-] NO analytics tracking.
